# 简答题

1. 复合矩阵中通常先缩放后旋转最后平移，不能互换是因为矩阵的乘法不满足交换律，旋转要放在平移之前是因为，平移后与旋转轴的位置会与平移前有所不同，那么若是先平移的话，就算是同样的旋转，平移的位置不同也会造成最后旋转的结果不同。缩放和旋转其实也差不多，比如让一个正方体乘一个x轴伸长一倍的矩阵，本意是想让它x轴的边伸长两倍成为长方体，但假若先绕x轴旋转，让对角边到达x轴，后缩放，那么底面就会成为菱形，从而达不到预期效果。
2. 正交投影后画面会不符合透视原理，且相互平行的线也会保持平行，投影后大小不会因投影的距离而发生变化。而透视投影后画面符合透视原理，更符合现实，且相互平行的线最后都会汇聚成一个点，投影后的大小会与投影的距离有关。
3. 绘制透明物体时关闭深度写入就不会造成前面的物体完全覆盖后面的物体，也就是说透明物体不会因为各自覆盖而造成使用融合技术时像素不会累加到最终的混合颜色中。从后往前绘制是为了让前面的透明物体与后面的物体进行混合，先绘制后面的物体才能使源像素与后台缓冲区的目标像素进行混合。
4. 一般渲染顺序使先渲染非透明物体，再从后往前渲染透明物体。
